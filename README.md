# Arquitectura de Computadoras I — Primer Parcial

Sitio web de estudio para el primer parcial de Arquitectura de Computadoras I, Séptimo Semestre.

🔗 https://axelmarroquinn.github.io/arquitectura-primerparcial

## Contenido

El sitio cubre los siguientes temas:

1. **Tipos de procesadores** — Microprocesador, Microcontrolador y SoC
2. **ISA: CISC vs RISC** — ISA, microarquitectura, comparación y combinaciones típicas
3. **Arquitecturas de memoria** — Von Neumann, Harvard, Harvard Modificada, jerarquía de caché, memoria en MCU (Flash / SRAM / EEPROM), bancos y mapa de memoria
4. **Pipeline** — Etapas IF→ID→EX→MEM→WB y tipos de hazards (data, control, structural)
5. **Paralelismo y Taxonomía de Flynn** — ILP, TLP, DLP, Power Wall, SISD/SIMD/MISD/MIMD, IA en edge (TPU, NPU, TinyML)
6. **Ley de Amdahl** — Fórmula, ejemplo resuelto y calculadora interactiva
7. **Sistemas embebidos** — MCU vs SBC, tabla comparativa PICAXE / Arduino / Raspberry Pi
8. **Buses** — Datos, dirección, control y protocolos (UART, SPI, I2C)
9. **Conceptos clave** — MIPS, CPI, interrupciones vs polling, Watchdog, consumo energético (Race to Sleep vs Slow & Steady)

Incluye tres calculadoras interactivas: **Ley de Amdahl**, **MIPS** y **Bancos de memoria**.
